How to Add a New Instrument

  1. Open data/instruments.json
  2. Add a new entry like below:
{
  "title": "Violin",
  "description": "A bowed string instrument used in orchestras and solo performances.",
  "image": "images/violin.jpg",
  "audio": "audio/violin.mp3"
}
  1. Save your violin image to the images/ folder.
  2. Save the violin audio sample to the audio/ folder.
